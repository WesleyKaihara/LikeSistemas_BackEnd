### OBS: Arquivo com informações de acesso ao banco de dados é privado ( nodemon.json )

- criar arquivo nodemon.js e criar a conexão com os requisitos abaixo

``` json
{
  "env": {
    "MYSQL_USER": "",
    "MYSQL_PASSWORD": "",
    "MYSQL_DATABASE": "",
    "MYSQL_HOST": "localhost",
    "MYSQL_PORT": 3306
  }
}

```
